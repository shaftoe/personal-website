# Personal Website

Source code for [a.l3x.in](https://a.l3x.in) — Alexander Fortin's personal website and tech blog.

See <https://a.l3x.in/colophon> for more details.

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

## Lint & Validate

```bash
bun run lint
bun run validate
```
